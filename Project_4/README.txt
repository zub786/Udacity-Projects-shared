# How to run the code
Steps to run:

1. To enjoy this project you will have to log in on vagrant
2. After Login, change the present working directory by using this command [ "cd /vagrant/catalog" ]
3. After executing above mentioned command execute the models.py file to generate database with tables for this use this command [ "python models.py" ]
4. After executing above mentioned command execute the application.py file to run the application for this use this command [ "python application.py" ]
5. Now enjoy the application

# How to use application
Usage:

1. On the left pane, all available categories are showing
2. On the right pane, all latest items of the categories are showing
3. If user is logged in he/she can do everything means Create, Update and Delete
4. But, Logged out user can only view the categories and there items by clicking on them.
